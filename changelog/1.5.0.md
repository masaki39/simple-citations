# Simple Citations 1.5.0

## New Features

### Multi bibliography support
- Multiple Better CSL JSON files can now be registered in settings.
- Files are prioritized by order; earlier entries take precedence when the same citation key appears in multiple files.
- Per-property merge strategies: choose **Priority** (use highest-priority file) or **Merge** (combine values from all files) for each field. `collections` and `library` default to Merge.
- A new **Bibliography** property (opt-in) records which file(s) each citation was sourced from.

### Mobile support
- Settings UI is now responsive and usable on mobile devices.

### New commands
- **Sync citations** — adds new notes and updates existing ones in a single step.
- **Reset all properties (Full sync)** — clears and rewrites all frontmatter properties; shows a confirmation modal before running.
