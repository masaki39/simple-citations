# Simple Citations 1.6.1

## Bug Fixes

### BetterBibTeX JSON: year parsing for MM/YYYY date format
- Fixed an issue where dates in `MM/YYYY` format (e.g. `"07/2019"`) were parsed as the month number instead of the year.
- The `year` property is now correctly extracted from all common date formats found in BetterBibTeX JSON exports: `YYYY`, `YYYY-MM`, `YYYY-MM-DD`, `YYYY/MM/DD`, and `MM/YYYY`.
