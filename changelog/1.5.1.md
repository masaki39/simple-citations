# Simple Citations 1.5.1

## Bug Fixes

### Zotero select URL for group libraries
- Fixed incorrect Zotero select URL generation for items in group libraries.
- Previously, group library items used the personal library URL format (`zotero://select/items/@key`), which did not open the correct item in Zotero.
- Now correctly generates `zotero://select/groups/{groupId}/items/@key` when the item belongs to a group library.
